# Onlyvip-Anti-Aids

this is to prevent the autorestart from restarting the whole process if for example 1 account gets kicked. it will wait until all of them are kicked or when the dungeon is done and then it will let autorestart restart. put this in your autoexecute folder in synapse x




# Usage

You HAVE to put your account name(s) in local accTable = {"accs"}, accs being your account name, DONT take out the "". local accTable = {"acc1", "acc2", "acc3"} is for having multiple accounts.
